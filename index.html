Html/Css

HTML e CSS são linguagens fundamentais para criar sites: o HTML (HyperText Markup Language) constrói a estrutura e o conteúdo (textos, links, imagens), como o esqueleto de um corpo, enquanto o CSS (Cascading Style Sheets) cuida da estilização e do design (cores, fontes, layout), sendo a "pele e roupa" que tornam a página visualmente atraente e organizada em diferentes dispositivos.

Ver mais

JavaScript

JavaScript (JS) é uma linguagem de programação versátil usada para criar interatividade e comportamento dinâmico em páginas web, permitindo elementos como animações, mapas interativos e atualização de conteúdo sem recarregar a página, atuando junto com HTML (estrutura) e CSS (estilo) como uma das tecnologias fundamentais da web.

Ver mais

WordPress

WordPress é um Sistema de Gerenciamento de Conteúdo (CMS) de código aberto, gratuito e poderoso, usado para criar e gerenciar sites, blogs, lojas virtuais e mais, sem precisar programar, graças à sua interface intuitiva, temas personalizáveis e plugins que adicionam funcionalidades, sendo a base de mais de 40% dos sites na internet e ideal para iniciantes e profissionais.
